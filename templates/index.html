📜 Game Rules

1. Each player places 3 pieces in turn.
2. Once each has 3, new moves replace the oldest (Fade-and-Replace phase).
3. First to align 3 wins!

ℹ️ About This Game

This is a web-based AI-enhanced Tic-Tac-Toe variant with fading moves,
built using HTML, CSS, JavaScript, and Python (Flask).
